# Sifra Tech Enterprise Web Portal

This repository contains the source code for the official Sifra Tech web portal. It is a modern, high-performance web application designed for enterprise scalability.

## Technology Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber

## Getting Started

### Prerequisites
Ensure Node.js and npm are installed in your environment.

### Installation
Clone the repository and install dependencies:

```bash
git clone https://github.com/VaishalMalu/Sifra-Tech-Website.git
cd sifratech-web
npm install
```

### Development Server
Run the development server locally:

```bash
npm run dev
```

Open http://localhost:3000 in your browser to view the application.

## Project Structure
- src/app: Contains the Next.js App Router configuration and primary layouts.
- src/components: Reusable React components for the user interface.
- src/assets: Static assets and images. Note that large internal design downloads are excluded from version control.

## Deployment
The application is configured for standard deployment on platforms supporting Next.js, such as Vercel. Refer to the standard Next.js deployment guidelines for specific environment variables and build configurations.
